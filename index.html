Technical Architecture

Guardian AI uses an advanced edge AI architecture designed for reliability, privacy, and affordability.

System Architecture

Hub

Raspberry Pi 4 Model B (4GB) running all inference & orchestration

Room Nodes

ESP32-CAM units (one per room) + optional Pi Camera for hub room

Audio

USB mic array at hub; optional I2S MEMS mics in rooms

Communication

Wi-Fi for room nodes ‚Üí hub; GSM (SIM800L) for offline alerting

Note on Technical Architecture: The Raspberry Pi + ESP32-CAM architecture described here represents our MVP development platform. For commercial production, we have designed and are building a custom integrated board that combines all these functionalities into a single, optimized hardware solution with enhanced reliability, lower power consumption, and reduced manufacturing costs.

üè≠

Production Hardware Evolution

While the Raspberry Pi + ESP32-CAM architecture served as our effective MVP development platform, we have engineered a custom System-on-Chip (SoC) solution for mass production that delivers superior performance, reliability, and cost efficiency.

üöÄ

Custom SoC Architecture

  • Processor: Quad-core ARM Cortex-A53 @ 1.5GHz with integrated Neural Processing Unit (NPU) capable of 3 TOPS for real-time multi-model AI inference
  • Memory: 2GB LPDDR4 RAM + 16GB eMMC flash with wear-leveling for continuous operation
  • Vision Processing: Dual MIPI CSI-2 camera interfaces with integrated Image Signal Processor (ISP) supporting HDR, noise reduction, and lens correction
  • Connectivity: Integrated 4G/LTE Cat-1 modem with fallback to 2G, dual-band Wi-Fi 6, Bluetooth 5.2 for peripheral connectivity
  • Audio Processing: 4-channel MEMS microphone array with beamforming and acoustic echo cancellation
  • Power Management: Advanced PMIC with 85% efficiency, support for 5-24V input range, and supercapacitor backup for 30-minute emergency operation
  • Security: Hardware security module with AES-256/SHA-2 acceleration, secure boot, and tamper detection
  • Environmental: Industrial temperature range (-20¬∞C to 70¬∞C), 6-layer PCB with EMI shielding

Manufacturing Advantages

  • Cost Reduction: 45% lower BOM cost compared to discrete component approach
  • Reliability: MTBF >100,000 hours with industrial-grade components
  • Power Efficiency: 60% lower power consumption (3W typical vs 8W for Pi-based solution)
  • Form Factor: 70% smaller footprint enabling discreet installation
  • Supply Chain: Single-board design reduces assembly complexity and component sourcing risks
